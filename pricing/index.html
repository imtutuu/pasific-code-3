Outlet is a store while register is a cashier machine or in other words, cash register or Checkout lane. One outlet may have one or multiple registers.

You do not need multiple registers on Hike if all staff users are logged into a same physical checkout register. Multiple users can login to the same register, that’s not a problem, if they are all on the same device. If your staff is using multiple devices to process sale transactions, the number of registers on Hike you require = total number of devices used for processing sales.

Yes, but not simultaneously.

Hike is on cloud, so you can logon to a register using any device and from anywhere. However, we do not advise selling on the same register from multiple devices at the same time. One register should always have one device. Using same register in multiple devices at the same time will result in duplicate receipt/invoice numbers.

Yes, certainly you can. If you like to change between devices for your register, simply close the register on current device, logon to another device and open the register there.

It is important, however, that you close the register in current device first to ensure that all transactions are uploaded to the cloud and in turn sent to the new device once you open the register there.
